Learning Check 544.10.1

Consider the null hypothesis (that there is no gender-based difference in promotion rates) under three different alpha values: 0.001, 0.01 and 0.05. What inferences would you make about the proportion of promotions under each of these alpha values? Compare the analytic approach (using hypothesis testing and p-values) to the visual approach (using confidence intervals). Describe the difficulties in interpreting the results.

When the value of alpha is 0.001, it is less than the p-value (p-value = 0.035).Similar is the case when the value of alpha is 0.01. In both of these cases we can reject the null hypothesis and we can say that the result is statistically significant. But when the value of alpha is 0.05 it is greater than the p-value and hence we fail to reject the null hypothesis and the result is statiscally non-significant. But since twice the value of alpha is less than p-value but once it is significantly greater than the p-value also it is difficult to decide which one to consider.
